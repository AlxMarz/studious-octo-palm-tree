PiNas - Raspberry Pi NAS Build
Project Overview

Built a robust home NAS server using Raspberry Pi 4 with OpenMediaVault, featuring RAID1 redundancy and Docker containerization for self-hosted services.

Technologies Used
Raspberry Pi 4 OpenMediaVault Docker Portainer RAID1 Linux
Key Features
  • 3x 1TB drives in RAID1 configuration for redundancy
  • Network file sharing via SMB/CIFS and NFS
  • Docker containerization with Portainer web UI
  • Automated backups and monitoring
  • Security hardening (SSH keys, Fail2Ban, firewall)
Challenges & Solutions

Implemented comprehensive security measures including SSH key authentication, Fail2Ban for intrusion prevention, and UFW firewall configuration. Optimized for 24/7 operation with proper cooling and UPS backup.
