PiNas - Raspberry Pi NAS Build
Overview

Built a robust home NAS server using Raspberry Pi 4 with OpenMediaVault, featuring RAID1 redundancy and Docker container. 3D printed case to house the Raspberry Pi and SATA Hat.

Technologies Used
Raspberry Pi 4 OpenMediaVault Docker RAID Linux
Key Features
  • 3x 1TB drives in RAID configuration for redundancy
  • Network file sharing via SMB/CIFS and NFS
  • Docker containerization with Portainer web UI
  • Automated backups and monitoring
Challenges & Solutions

Learning linux CLI and troubleshooting hardware compatibility issues with SATA hat fitting the case. Reduced the size of the heat sink to fit the housing. Using OMV and docker documentation to set up the NAS and containerization.
